## 👨‍💻 About Me

복잡한 비즈니스 문제를 보안, 데이터, 인프라, 실시간 통신, 이벤트 스트리밍, AI Agent의 6개 연구 축으로 분해하고, 각 축을 실행 가능한 레퍼런스 코드와 기술문서로 증명하는 AI FullStack Developer입니다.

- **Architecture**: 도메인 경계, 운영 안정성, 확장 가능한 시스템 구조를 설계합니다.
- **Data & Backend**: 데이터 접근 전략, 트랜잭션, 성능 최적화를 코드와 테스트로 검증합니다.
- **AI Engineering**: RAG, LLM, Agent workflow를 백엔드 시스템 안에 통합하는 방식을 연구합니다.
